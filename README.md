# IRO – Issues Reporting Outlet

## Overview
IRO (Issues Reporting Outlet) is a web-based platform designed to help AAUA students and indigenes of Akungba, Iwaro, Etioro, and Ayegunle report and track community or campus-related issues. This project was developed as part of CSC426 (Weekend 3 Assignment).

## Problem Statement
There is currently no formal, centralized channel through which students and indigenes can report issues such as poor infrastructure, insecurity, electricity/water supply problems, or campus-related concerns. IRO addresses this gap with a simple web-based reporting system.

## Features
 Submit an issue report (name, respondent type, community, issue type, description)
View a list of all reported issues
Track issue status (Open / Resolved)
 Mark an issue as resolved (admin)

## Tech Stack
- HTML5
- CSS3
- JavaScript (Vanilla)
- Browser Local Storage (for demo data persistence)
